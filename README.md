## my_card_flutter

This is a Profile Card which includes a basic sample profile UI:
- Profile Image
- Name
- Job/Position title
- Phone Number
- Email

## Getting Started

This Project is a small App created in Flutter to better understand the basic Layout Widgets such as Column, Rows etc.
And some built-in easy-to-use widgets were used alternative to the basic 
widgets such as, ListTile and Card Widgets.
